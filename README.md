# Tableau Dashboards Portfolio

## Overview
This repository contains interactive Tableau dashboards created to explore sales and market trends through data visualization. The projects focus on identifying patterns, comparing categories, and presenting insights in a clear and accessible format.

---

## Projects

### 1. Bike Sales Dashboard (Sample Data)
This dashboard uses sample data to demonstrate data visualization and dashboard design skills in Tableau.

**Key focus areas**
- Sales trends
- Category-level comparisons
- Product performance
- Customer insights

**Tableau Public**
[View the Bike Sales Dashboard](https://public.tableau.com/app/profile/risa.kuriyama/viz/BikeSales_17271360628120/SalesDashboad)

---

### 2. EV Sales Analysis (Washington State)
This dashboard analyzes electric vehicle (EV) data in Washington State, focusing on adoption trends and key patterns in the local EV market.

**Key focus areas**
- EV adoption trends over time
- Distribution of EV types
- Growth patterns in the Washington EV market
- Comparative insights across categories

**Tableau Public**
[View the EV Sales Dashboard](https://public.tableau.com/app/profile/risa.kuriyama/viz/WashingtonEVDashboad/Dashboard2)

## プロジェクト

### 1. Bike Sales Dashboard（サンプルデータ）

このダッシュボードは、サンプルデータを用いてTableauにおけるデータ可視化スキルおよびダッシュボード設計力を示すことを目的としています。

**主なポイント**

* 売上トレンド
* カテゴリ別の比較
* 商品ごとのパフォーマンス
* 顧客に関する分析

**Tableau Public**
[Bike Sales Dashboardを見る](https://public.tableau.com/app/profile/risa.kuriyama/viz/BikeSales_17271360628120/SalesDashboad)

### 2. EV販売分析（ワシントン州）

このダッシュボードは、ワシントン州における電気自動車（EV）のデータを分析し、普及の動向やローカル市場における特徴を可視化したものです。

**主なポイント**

* EV普及の時系列トレンド
* EVタイプの分布
* ワシントン州における市場成長の傾向
* カテゴリ間の比較分析

**Tableau Public**
[EV Sales Dashboardを見る](https://public.tableau.com/app/profile/risa.kuriyama/viz/WashingtonEVDashboad/Dashboard2)
